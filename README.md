# MM-ForecastIO

## Dependencies (Add following packages to package.json)

"d3":"latest"

"jsdom": "latest"

## Installation

- Clone this repo
- Rename the folder from "MM-ForecastIO" to forecastio
- put the folder in MagicMirror modules directory

## config

{

  module: "forecastio",
  
  position: "top",
  
  config:{
  
     appkey:"APPKEY",
     
     latitude: 43.6529,
     
     longitude: -79.3849
     
  }
  
}
